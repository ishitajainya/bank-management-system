In-House Bank Management Application

A secure internal banking system developed using Spring Boot and Angular to manage daily banking operations with role-based access control and approval workflows.

Overview

This application is designed for internal bank staff (Manager and Clerk). It is not a customer-facing system.

Key Features

-> Role-based authentication using JWT

-> Deposit and withdrawal operations

-> Manager approval for high-value withdrawals

-> Immutable transaction history for audit trail

Technology Stack

-> Backend: Java 17, Spring Boot, Spring Security, JPA

-> Frontend: Angular, TypeScript

-> Database: MySQL

-> API Documentation: Swagger

Future Scope

-> Fraud detection using Machine Learning

-> Enhanced audit reporting

-> Cloud deployment and scalability
